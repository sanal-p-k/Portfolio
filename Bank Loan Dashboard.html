Project detail

This project focuses on analyzing and visualizing loan-related data to derive actionable insights and support decision-making in the financial domain. I worked with a dataset containing loan attributes such as loan amounts, interest rates, grades, employment details, and borrower information for the year 2024.

Key Contributions:

Data Preparation using SQL:

* Extracted, cleaned, and transformed raw data stored in a relational database to prepare it for analysis.

* Wrote complex SQL queries to filter, aggregate, and join tables, ensuring the dataset was structured and comprehensive for visualization.

* Performed calculations like debt-to-income ratio (DTI), total payments, and interest summaries directly in SQL for efficiency.

Interactive Dashboards in Tableau:

* Designed and developed interactive dashboards to provide stakeholders with a clear understanding of loan performance and borrower demographics.

* Visualized key metrics, including loan distribution by state, grade, and purpose, using bar charts, heatmaps, and scatter plots.

* Created calculated fields within Tableau to dynamically display borrower risk profiles and repayment behavior.

* Implemented filters and drill-down capabilities, enabling users to explore data by loan grade, employment status, and regional trends.

Outcome:

This project provided actionable insights into borrower trends, repayment behavior, and regional loan demand, enabling data-driven strategies for loan management and marketing. The interactive dashboards streamlined the decision-making process for financial analysts and stakeholders.
