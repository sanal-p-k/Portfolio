Project Detail

Designed and developed an interactive Real Estate Power BI Dashboard to visualize and analyze property data effectively. The dashboard includes dedicated pages for Home, Sold Properties, and Rental Listings, offering real-time insights into property status, pricing trends, and regional distribution. With a clean, user-friendly interface and powerful filters, this tool helps users and stakeholders make informed decisions based on current market data.
